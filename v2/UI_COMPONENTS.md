# UI Компоненты v2.0

## Структура

Все UI компоненты вынесены в отдельную папку `ui/` для модульности и переиспользования.

## Компоненты

### ChatUI (`ui/chat.js`)
Управляет чатом и сообщениями.

**Методы:**
- `addMessage(text, type, isGlitch)` - добавить сообщение
- `showTyping()` / `hideTyping()` - показать/скрыть индикатор печати
- `showChoices(choices)` - показать варианты выбора
- `showMessages(messages, index, callback)` - показать последовательность сообщений

### StatsUI (`ui/stats.js`)
Экран детальной статистики.

**Показывает:**
- Основные параметры (стабильность, настроение, энергия, доверие, любопытство)
- Черты личности
- Прогресс (день, воспоминания, записи в дневнике, достижения)
- Активные цели с прогрессом

### ActivitiesUI (`ui/activities.js`)
Экран выбора активностей.

**Функции:**
- Отображение доступных активностей в виде карточек
- Показ эффектов каждой активности
- Выполнение активности с анимацией реакции
- Автоматическое создание воспоминаний

### DiaryUI (`ui/diary.js`)
Дневник Ани.

**Показывает:**
- Все записи в дневнике (новые сверху)
- Дату и настроение для каждой записи
- Текст записи

### AchievementsUI (`ui/achievements.js`)
Система достижений.

**Функции:**
- Отображение всех достижений (открытых и закрытых)
- Счётчик прогресса
- Всплывающие уведомления при получении достижения
- Метод `showNotification(achievement)` для показа уведомления

### MemoriesUI (`ui/memories.js`)
Альбом воспоминаний.

**Показывает:**
- Последние 20 воспоминаний
- Тип воспоминания (иконка)
- Важность (звёздочки)
- Текст и дату

### SecretsUI (`ui/secrets.js`)
Секреты Ани.

**Функции:**
- Отображение секретов по уровням доверия
- Три состояния: закрыто, доступно, раскрыто
- Кнопка раскрытия секрета
- Автоматическое создание воспоминания при раскрытии

## Использование в game.js

```javascript
// Инициализация
this.chatUI = new ChatUI(this);
this.chatUI.init();

// Использование
this.chatUI.addMessage('Привет!', 'anya');
this.activitiesUI.show();
this.statsUI.show();
```

## Стили

Все стили компонентов находятся в `styles/components.css`:
- Модальные окна
- Карточки активностей
- Записи дневника
- Достижения
- Воспоминания
- Секреты
- Уведомления

## Интеграция

Все компоненты подключаются в `index.html` после core систем:

```html
<!-- UI компоненты -->
<script src="ui/chat.js"></script>
<script src="ui/stats.js"></script>
<script src="ui/activities.js"></script>
<script src="ui/diary.js"></script>
<script src="ui/achievements.js"></script>
<script src="ui/memories.js"></script>
<script src="ui/secrets.js"></script>
```

## Преимущества

1. **Модульность** - каждый компонент независим
2. **Переиспользование** - легко использовать в разных местах
3. **Поддержка** - легко найти и исправить баги
4. **Расширяемость** - легко добавить новые компоненты
5. **Чистый код** - game.js не перегружен UI логикой
